# 	‘A’和“A”有什么区别？
- 'A' 单引号里面是字符A，占用一个字节
- "A" 双引号里面是字符串A, 由于字符串本质是以'\0'结尾的**字符数组**，占用两个字节，  
  | 0 | 1 |
  | ---- | ---- |
  | A | \0|
